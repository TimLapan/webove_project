История создания языка Python

Краткая хронология основных событий, связанных с разработкой и развитием Python.

1980-е годы

Гвидо ван Россум начал работу над Python в конце 1980-х годов. Целью было создать язык, простой в использовании.

1991 год

Первый выпуск Python 0.9.0 с поддержкой классов, исключений и модулей.

2000 год

Выпуск Python 2.0, добавивший управление памятью и поддержку Unicode.

2008 год

Выпущен Python 3.0 с улучшенной обработкой текста.

2020 год

Окончание поддержки Python 2, переход на Python 3.
